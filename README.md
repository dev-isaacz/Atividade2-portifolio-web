# Atividade 2 - Portfólio Web
 
Primeira versão de um site de portfólio pessoal, desenvolvida com **HTML e CSS**, como atividade prática durante o curso Transforme-se em Desenvolvimento Front-End (Serasa Expirence & Proa). O projeto apresenta informações pessoais, habilidades e projetos de forma estruturada.
 
## Tecnologias
 
- HTML5
- CSS3
  
## Funcionalidades
 
- Seção de apresentação pessoal
- Seção de habilidades/tecnologias
- Layout estruturado com HTML semântico

## Aprendizados
 
Atividade do curso de Front-End para praticar:
- Estruturação de uma página de portfólio
- HTML semântico
- Estilização com CSS
